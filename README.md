# The source code for johnsilvell.com
The official source code for the brand new design for www.johnsilvell.com

## Fonts
All the fonts are collected from https://fonts.google.com/.

## Styling
The stylling is primarly made with CSS and can be found in the file: "style.css"

## Script
For more dynamic functionality I've used JavaScript (like any sane web developer). The JS-code can be found in the file: "script.js".

## Bootstrap
To make the website responsive to different screens and devices I have used Bootstrap 5. The official source code for Bootstrap can be found on https://getbootstrap.com/. 

### Bootstrap CDN
Since I do not host https://www.johnsilvell.com/ on my own server I have included Bootstrap from a Content Delivery Network (CDN), jsDelivr.
